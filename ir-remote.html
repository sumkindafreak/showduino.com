Infrared Remote

Retro Control Meets Horror Tech

Overview

Take full control without touching the screen. Showduino's infrared remote system brings TV-style input to your haunt tech, ideal for prop operators, hidden tech panels, or wireless puzzle triggers.

Remote Capabilities

  • • Navigate menu items with arrow + OK buttons
  • • Trigger effects, sequences, or system actions wirelessly
  • • Use numeric entry for quick access to hidden features

Shift Functionality

Support for capital letters and symbols is built in using a SHIFT toggle on the remote. This allows full password input for WiFi, Bluetooth naming, and more — all without a keyboard.

Perfect for Escape Rooms

IR remote control is ideal for operators running shows from behind the scenes. It also allows you to embed tech control into set pieces, puzzle props, or even wearable gear.

Planned Upgrades

  • • Custom remote profiles with learnable buttons
  • • Configurable trigger sequences
  • • Infrared recording and playback for puzzle integration
