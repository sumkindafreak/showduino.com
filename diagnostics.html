Diagnostics + Self-Test

Because Even Monsters Need Maintenance

Overview

Stay one step ahead of the chaos. Showduino's diagnostics engine gives you real-time hardware feedback and runs scripted tests to ensure every haunt element is ready for action.

Live Data Checks

  • • Bluetooth + WiFi status
  • • SD card presence + health
  • • Temperature + voltage readings (from Mega)
  • • NeoPixel group brightness levels
  • • Relay & input pin states

Manual Control

All diagnostics are triggered manually — no automatic updates to avoid clutter or CPU drag. Just tap the button when you need the truth.

Self-Test Mode

Launch a full self-test sequence from the menu. It checks connections, toggles relays, queries inputs, and confirms visual and audio output — all with pass/fail feedback.
